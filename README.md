# 🛡️ Forensicator Documentation

Official documentation and detection guides for **Forensicator** — a host-based forensic analysis and threat detection tool.

---

## 📖 Overview

This repository contains structured documentation for findings generated by Forensicator.

Each page provides:

* Detection context
* What the finding means
* What to investigate
* Potential security risks
* Recommended next steps

The goal is to provide **clear, actionable insight** without relying solely on automated interpretation.

## 🔍 What You’ll Find

Each detection page includes:

### ✅ Summary

A high-level explanation of the finding.

### ⚠️ Why It Matters

Security impact and potential attack scenarios.

### 🔎 What to Look For

Key indicators such as:

* Unusual patterns
* Repeated activity
* Suspicious sources

### 🧪 Investigation Steps

Practical steps to validate or escalate the finding.

### 🎯 MITRE ATT&CK Mapping

Where applicable, detections are mapped to known techniques.

### ❗ False Positives

Common benign scenarios to avoid misinterpretation.

## 🔗 Usage in Forensicator Reports

Findings generated by Forensicator include links to this documentation.

Example:

```html
<a href="https://docs.forensicator.com/detections/users-accounts" target="_blank">
  Learn more
</a>
```

This allows analysts to quickly pivot from raw data to **contextual understanding**.
